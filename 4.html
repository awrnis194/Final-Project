ALL STAR CODE

Mission: To create economic opportunity by developing a pathway for young boys to enter the tech and computer world.

Vision: To see a country where young men of color have the ability to succeed and to strive in life.

What is All Star: All Star Code is a 6 week intensive in which you learn different coding languages while gaining skills to help you in your proffessional life. All Star Code is good for students that want a step towards a coding career but dont know where they want to start.
