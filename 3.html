All Star Code

All Star Code is a free program directed towards young men of color that takes place for 6 weeks of the summer. It teaches different coding languages giving you a jump start to your coding future.



Lehman College Coding Camp

Hosted by Lehman College in the Bronx, this program allows students to see how college students live from a basic day to day life. Students learn different elements of computer programing and coding
